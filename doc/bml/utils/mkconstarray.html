mkconstarray

PURPOSE

MKCONSTARRAY creates a constant array of a specified numeric class.

SYNOPSIS

function out = mkconstarray(class, value, size)

DESCRIPTION 

MKCONSTARRAY creates a constant array of a specified numeric class.
   A = MKCONSTARRAY(CLASS, VALUE, SIZE) creates a constant array 
   of value VALUE and of size SIZE.
